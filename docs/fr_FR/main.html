Fenêtre principale de pgAdmin

Dans la fenêtre principale, la structure des bases de données est affichée. Vous pouvez créer de nouveaux objets, supprimer et modifier les objets déjà existants si les droits de l'utilisateur que vous utilisez pour la connexion en cours sur cette base le permettent.

Le côté gauche de la fenêtre principale affiche une structure en arbre de tous les serveurs et des objets qu'ils contiennent.

En haut à droite sont affichées les détails de l'objet sélectionner dans l'arbre. Certains objets ont des statistiques en plus de leur propriétés. Ces statistiques peuvent être affichées si vous sélectionnez l'onglet Statistiques.

En bas à droite se trouve le script SQL permettant la création de l'objet sélectionné. Vous pouvez le copier dans un éditeur en utilisant le copier/coller ou le sauvegarder dans un fichier en utilisant le menu Sauvegarder la définition... à partir du menu Fichier, ou l'utiliser comme un modèle si vous sélectionnez l'outil de requêtage. Si l'option Copier le SQL est sélectionner, la requête SQL sera automatiquement copiée dans l'outil.

La ligne de statut vous afficher quelques informations sur l'état, ainsi que le moment de la dernière action réalisée avec succès par pgAdmin III.

Vous pouvez redimensionner la fenêtre principale et modifier les tailles des trois régions principales si vous préférez. Ces ajustements seront préservés quand vous quitterez le programme.

pgAdmin fait attention à la bande passante utilisée. Le statut des objets dans le navigateur est seulement rafraichi par une demande utilisateur ou après modifications réalisées par les outils intégrés. Faites attention que cela ne couvre pas les modifications réalisées par des requêtes SQL manuelles, pas plus que cela ne couvre les modifications réalisées par d'autres utilisateurs et d'autres clients. Il est généralement préférable de rafraichir manuellement les objets avant de travailler sur eux dans de tels environnements.

Pour commencer

Après avoir ajouté les serveurs désirés dans le navigateur en utilisant le menu Ajouter un serveur ou le bouton de la barre d'outils, chaque serveur s'affichera dans le nœud principal, "Serveurs".

Pour ouvrir une connexion à un serveur, sélectionnez le serveur désiré dans le navigateur, double cliquez dessus ou utilisez Se connecter dans le menu Outils. La connexion est établie et les propriétés des objets principaux sont récupérés du serveur. Si vous avez déjà été connecté à cette base de données, pgAdmin III restaurera la sélection précédente de base de données et de schéma. La situation actuelle est sauvegardée à la sortie du programme pour que pgAdmin III soit capable d'y revenir.

En utilisant le menu ou les boutons de la barre d'outils, vous pouvez créer de nouveaux objets, supprimer et modifier les propriétés des objets existants si l'utilisateur utilisé pour la connexion au serveur a suffisamment de droits. Certains options peuvent être inactives lors de l'affichage des propriétés. Cela signifie que le serveur que vous utilisez ne supporte pas cette fonctionnalité ou que cette propriété ne peut pas être modifiée ou que vos droits ne vous le permettent pas.
