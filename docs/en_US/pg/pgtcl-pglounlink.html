pg_lo_unlink

Name

pg_lo_unlink -- delete a large object

Synopsis

pg_lo_unlink conn loid

Description

pg_lo_unlink deletes the specified large object.

Arguments

conn

The handle of a database connection in which the large object exists.

loid

The OID of the large object.

Return Value

None
