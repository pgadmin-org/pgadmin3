14.3. Getting The Source

The PostgreSQL 8.1devel sources can be obtained by anonymous FTP from ftp://ftp.postgresql.org/pub/source/v8.1devel/postgresql-8.1devel.tar.gz. Use a mirror if possible. After you have obtained the file, unpack it: 

gunzip postgresql-8.1devel.tar.gz
tar xf postgresql-8.1devel.tar

This will create a directory postgresql-8.1devel under the current directory with the PostgreSQL sources. Change into that directory for the rest of the installation procedure.
